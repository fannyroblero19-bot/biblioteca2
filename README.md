# Biblioteca System

Proyecto demo de sistema de biblioteca con frontend y backend separados.

## Estructura

biblioteca-system/
├── backend/
│   ├── server.js
│   ├── package.json
│   ├── database.js
│   └── routes/
│       ├── books.js
│       └── users.js
├── frontend/
│   ├── index.html
│   ├── style.css
│   └── script.js
└── README.md

## Requisitos

- Node.js
- npm

## Instalación

1. Abre terminal en `biblioteca-system/backend`
2. Ejecuta `npm install`
3. Ejecuta `npm start`
4. Abre `http://localhost:4000`

## Funcionalidades

- CRUD de libros
- CRUD de usuarios
- Eliminación de registros
- Almacenamiento con SQLite
- Conexión entre frontend y backend
- Diseño oscuro moderno con sidebar lateral
